##라이브러리와 프레임워크의 차이

라이브러리는 자주 사용하는 기능을 미리 만들어 놓은 API의 집합

프레임워크는 코드가 동작하는 규칙과 틀을 규정해주며 대표적인 역할은 생명주기 관리

- 나의 코드가 호출하면 라이브러리
- 나의 코드를 호출하면 프레임워크
- 프레임워크는 라이브러리를 포함
